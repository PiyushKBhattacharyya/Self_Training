Essential commands to diagnose network issues.

## Tools
1.  **ping**: Check if a host is reachable.
2.  **traceroute**: Trace the path to a host.
3.  **nslookup / dig**: Query DNS servers.
4.  **telnet / nc**: connectivity to specific ports.

## Examples
- `ping 8.8.8.8`: Ping Google DNS.
- `nslookup google.com`: Find IP of domain.
- `nc -vz google.com 80`: Check if port 80 is open.
