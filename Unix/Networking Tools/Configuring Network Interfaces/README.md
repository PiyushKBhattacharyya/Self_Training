Linux provides a suite of tools to configure and troubleshoot network interfaces.

## Key Tools
- **ifconfig** (Legacy): Configure network interfaces.
- **ip** (Modern): Show / manipulate routing, devices, policy routing and tunnels.
- **ping**: Check connectivity.
- **netstat**: Print network connections, routing tables, interface statistics.
- **ss**: Another utility to investigate sockets.

## Common Commands
- `ip addr show`: Display IP addresses.
- `ping google.com`: Test connection.
- `ss -tuln`: Show listening ports.
