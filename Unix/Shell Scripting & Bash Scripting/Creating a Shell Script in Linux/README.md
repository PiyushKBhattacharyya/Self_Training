## Steps
1.  **Create File**: Use `touch scriptname.sh` or an editor (`nano`, `vi`).
2.  **Shebang**: Add `#!/bin/bash` at top.
3.  **Permissions**: Make it executable using `chmod`.
    - `chmod +x scriptname.sh` (User/Group/Others execute)
    - `chmod 755 scriptname.sh` (Read/Write/Execute for Owner, Read/Execute for others)
4.  **Run**: `./scriptname.sh`

## Common Modes
- `vi script.sh`: Edit in Vi.
- `cat > script.sh`: Create from stdin.
