Unix is a powerful, multi-user, multi-tasking operating system originally developed at AT&T Bell Labs.

## Key Features
1.  **Multi-user**: Multiple users can use the system resources simultaneously.
2.  **Multi-tasking**: Capable of executing multiple programs.
3.  **Portability**: Can be moved to different hardware with minimal changes.
4.  **Security**: Permission-based file system.

## The Unix Philosophy
> "Write programs that do one thing and do it well. Write programs to work together. Write programs to handle text streams, because that is a universal interface." - Doug McIlroy
