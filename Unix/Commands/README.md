# ⌨️ Unix Commands

Key Unix commands for file management, process control, and system information.

## Universal Commands
- `ls`: List directory contents.
- `cd`: Change directory.
- `pwd`: Print working directory.
- `cp`: Copy files/folders.
- `mv`: Move/Rename files/folders.
- `rm`: Remove files/folders.
- `grep`: Search text using patterns.
- `chmod`: Change file modes (permissions).

## Example
```bash
ls -la
grep "error" logfile.txt
chmod +x script.sh
```
