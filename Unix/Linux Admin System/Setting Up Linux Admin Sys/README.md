General guidelines for setting up an administrative environment.

## Key Steps
1.  **User Management**: Create admin users with `sudo` access.
2.  **Network Config**: Set static IPs if necessary.
3.  **Package Updates**: Ensure system is up to date (`apt update` / `yum update`).
4.  **Security**: Configure firewall (`ufw` / `iptables`) and SSH keys.

## Checklist
- [ ] Root password secured.
- [ ] Sudo user created.
- [ ] SSH root login disabled.
- [ ] Firewall active.
