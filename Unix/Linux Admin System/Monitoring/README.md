Monitoring system resources is crucial for maintaining performance and diagnosing issues.

## Key Tools
1.  **top / htop**: Real-time process monitoring.
2.  **df**: Disk space usage (Disk Free).
3.  **du**: Disk usage of files/folders.
4.  **free**: Memory usage.
5.  **vmstat**: Virtual memory statistics.
6.  **iostat**: CPU and I/O statistics.

## Examples
- `top`: Interactive process viewer.
- `df -h`: Human-readable disk space.
- `free -m`: Memory in MB.
