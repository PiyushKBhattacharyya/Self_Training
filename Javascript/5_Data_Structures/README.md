## Arrays
Ordered lists of values.
- **Methods**: `push`, `pop`, `map`, `filter`, `reduce`.

## Objects
Key-value pairs.
- **Access**: Dot notation (`obj.key`) or Bracket notation (`obj['key']`).

## Example
```javascript
const users = [
    { id: 1, name: "Alice" },
    { id: 2, name: "Bob" }
];

const names = users.map(u => u.name);
// ["Alice", "Bob"]
```
