Modern Javascript features introduced in ECMAScript 6 and later.

## Key Features
1.  **Destructuring**: Extracting values from arrays/objects.
2.  **Spread Operator (`...`)**: Expanding iterables into elements.
3.  **Rest Parameters (`...`)**: Gathering arguments.
4.  **Template Literals**: String interpolation with backticks.

## Examples
```javascript
// Destructuring
const { name, age } = user;

// Spread
const combined = [...arr1, ...arr2];

// Template Literal
console.log(`Hello, ${name}!`);
```
