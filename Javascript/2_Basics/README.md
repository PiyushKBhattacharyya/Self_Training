## Variables
- `var`: Function scoped (Older, avoid if possible).
- `let`: Block scoped (Modern, use for variables that change).
- `const`: Block scoped (Modern, use for constants).

## Data Types
1.  **String**: Text (`"Hello"`, `'World'`).
2.  **Number**: Integers and floats (`42`, `3.14`).
3.  **Boolean**: `true`, `false`.
4.  **Undefined**: Variable declared but not assigned.
5.  **Null**: Intentional absence of value.

## Syntax
```javascript
let name = "Alice";
const PI = 3.14159;
```
