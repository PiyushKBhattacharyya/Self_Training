## Conditionals
- `if`, `else if`, `else`.
- `switch`.

## Loops
- `for`: Classic loop.
- `while`: Runs while condition is true.
- `do...while`: Runs at least once.

## Comparators
- `==`: Value check (loose equality).
- `===`: Value and Type check (strict equality). **Recommended**.

## Example
```javascript
if (age > 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}
```
