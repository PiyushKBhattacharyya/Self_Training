## Conditionals
Using `if`, `elif`, and `else`. Remember the colons!
```python
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("C")
```

## Loops
1.  **For Loop**: Iterate over sequence.
    ```python
    for i in range(5): # 0 to 4
        print(i)
    ```
2.  **While Loop**: Run while true.
    ```python
    while x > 0:
        x -= 1
    ```
