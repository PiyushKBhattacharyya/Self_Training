## Modules
A file containing Python code (functions, classes) that can be imported.
```python
import math
print(math.sqrt(16))
```

## Packages
A directory containing multiple modules and a `__init__.py` file (optional in Python 3.3+).

## Standard Library
Python comes with "Batteries Included":
- `math`: Mathematical functions.
- `datetime`: Date and time.
- `os`: Operating system interface.
- `random`: Random number generation.

## Pip
The package installer for Python (used to install libraries like `requests`, `numpy`).
`pip install requests`
