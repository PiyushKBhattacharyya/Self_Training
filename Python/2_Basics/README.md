## Variables
No declaration needed. Just assign value.
```python

name = "Python"
```

## Data Types
1.  **int**: `10`, `-5`
2.  **float**: `3.14`
3.  **str**: `"Hello"`, `'World'`
4.  **bool**: `True`, `False`

## Input/Output
- `print()`: Output to console.
- `input()`: Read string from user.

## Type Casting
- `int("10")` -> `10`
- `str(10)` -> `"10"`
